ARCHIVES EXAMPLE1: UITS Starlab Project

SUMMARY

Example interface to Starlab data, to learn how to setup a database and query it. Plummer, King_w=7 and King_w=9 models with N=1024, 2048 and 4096 were generated, and ran for 10 time units with the kira integrator. Log file, Input and Output data are retrievable.
