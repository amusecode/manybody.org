MODEST: Links to Related Sites

  • AstroComp, a portal allowing the use of parallel computer codes for astrophysical and cosmological simulations, along with allocations of CPU time on parallel machines.
  • NEMO, a software environment for collisionless stellar dynamics, with tools to set up, integrate and analyze N-body datasets.
  • SCYON, a bi-monthly newsletter devoted to star cluster research.
  • Starlab, a software package for simulating, analyzing, and visualizing the evolution of dense stellar systems.
  • Sverre Aarseth's N-Body download page. Everything you need to know about N-body simulations!
